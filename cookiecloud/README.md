# CookieCloud

源项目: <https://github.com/easychen/CookieCloud>

- CookieCloud是一个和自架服务器同步Cookie的小工具，可以将浏览器的Cookie及Local storage同步到手机和云端，它内置端对端加密，可设定同步时间间隔。
- container-station >= 2.0
